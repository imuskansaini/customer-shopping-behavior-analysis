# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Customer Shopping Behavior Dataset (3900 records)

## Analysis Performed
- Data inspection
- Data cleaning
- Missing value analysis
- Feature engineering
- Customer age segmentation
- Purchase frequency mapping
- Average purchase by gender
- Average purchase by age group
- Average purchase by category

## Key Insights
- The dataset contained no missing values.
- Middle-aged customers had the highest average purchase amount.
- Average spending across product categories was fairly similar.
- Redundant columns were identified and removed.
